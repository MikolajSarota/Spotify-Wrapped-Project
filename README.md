Spotify-Wrapped-Project

Goal:
Recreate Spotify Wrapped using your entire listening history.

Rationale:
Spotify Wrapped is limited to yearly snapshots. This project extends the idea by analyzing long-term listening behavior, habits, and preferences using raw streaming data.

Technology: 
Python, Pandas, JSON data (Spotify Extended Streaming History), Power BI (for visualization)

Outcomes:

Total listening time and play count,
Most listened songs and artists,
Morning vs evening listening behavior,
Longest listening streak,
Most frequently clicked songs,
Seasonal listening patterns,
Power BI visualisation,
